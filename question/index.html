Question-1: What are the different ways to select an element in the DOM?


Ans: getElementById(): Select an element by its id, getElementByClassName(): Selects element by their class name, getElementByTagName(): Selects elements by their tag name, getElementsByName: Selects elements by their name arrribute, querSelectorAll(): Selects all elements matching a css selector, querySelector(): Selects the first element matching a css selector, closest(): Finds the nearest ancestor matching a css selector.

Question-2: What is the difference between innerHTML, innerText, and textContent?


Ans: innerHTML: Sets HTML content, including tags, innerText: Sets visible text, considering css style, textContent: Sets all text, no tags.

Question-3: What is event delegation in the DOM?


Ans: Event delegation is when you attach a single event listener to a parent element, and it handles events from child elements through event bubbling. This reduces the number of listeners and works well for dynamically added elements.

Question-4: What is event bubbling in the DOM?


Ans: Event bubbling is when an event starts at the target element and bubbles up to its parent elements. You can stop it with event.stopPropagation(). It helps with event delegation and managing events efficiently.
